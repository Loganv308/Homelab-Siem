# Homelab-SIEM

This repository contains all SIEM (Security Information and Event Management) related configurations, scripts, and resources running in my homelab. It serves as a central hub for monitoring, logging, alerting, and security analysis.

---

## Overview

This repo is designed to centralize all SIEM-related infrastructure in a homelab environment. It aims to:

- Provide a modular and reproducible setup
- Enable monitoring, logging, and alerting across systems
- Support multiple SIEM tools for experimentation and learning
- Document configurations for easy replication

---

## Architecture

The SIEM stack in this homelab includes:

- **Log Collection:** Agents on endpoints and VMs
- **Log Aggregation:** Centralized log storage
- **Analysis & Alerting:** Rule-based and statistical alerts
- **Visualization:** Dashboards for logs, metrics, and alerts
